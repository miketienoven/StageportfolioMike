Matrix test scene


Ik heb voor het onderzoek in een aantal programma's die ik heb vergeleken zelf een test scene gemaakt. Ik heb gekozen voor deze specifieke scene, omdat het een gesloten ruimte (gang) is waar alleen de hoofdpersoon en de hoofdschurken in de shots voorkomen. Ik heb in het programma gebruik gemaakt van de roombuilder functie, hiermee kan ik op een plattegrond muren tekenen en deuren tekenen en dan maakt het programma hier een ruimte van. Dit werkt erg prettig en snel. vervolgens kan ik ook op de plattegrond personages plaatsen en deze door er 3x op te klikken emoties via een menu aanklikken.
